# The Problem

In yay, after you upgrade a package, it is up to you to rebuild it when needed (https://bbs.archlinux.org/viewtopic.php?id=294674). This is the case for sioyek.

# The solution

The command to rebuil sioyek is
```
yay --rebuild sioyek
```
